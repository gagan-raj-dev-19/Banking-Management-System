# Banking Management System

A simple **Core Java OOP-based console application** that simulates basic banking operations.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Transfer money
- View balance

## OOP Concepts Used
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Interfaces
- Exception Handling

## Project Structure
- model
- service
- transaction
- exception
- util
- app

## How to Run
- Open project in IntelliJ IDEA
- Run `MainApp.java`
